### data types [mdn link](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures)

- Boolean
- Null
- Undefined
- Number
- String
- Symbol (new in ECMAScript 6)

we shall see boolean, string and number for now



``` javascript
// numbers
console.log(1)
console.log(-1)
console.log(1.234)

// boolean
console.log(true)
console.log(false)

// strings - can be in single or double quotes
console.log('hello')
console.log("hello")
```

*escaping in strings*

```javascript
console.log('abc\'def');
console.log("abc\'def");
```
